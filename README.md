## dependencies
Tested with Python 3.6.
Install dependencies with `pip install -r requirements.txt`.

## for the server

The start the development server:

```bash
python app.py
```

or for debug mode:

```
cd src
python app.py -d
```

### for the client

To start the React client:

```
cd client
npm i
npm start
```
